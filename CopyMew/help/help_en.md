#CopyMew - Help

##Action list

shortcuts | function
--------|-------------
 ↑      | Select the previous item	
 ↓      | Select next item 
 ←      | Go to the previous Pinboard
 →      | Go to the next Pinboard
 ↵      | Paste selected item
space   | Preview
esc	     | Hide Paste
delete	 | Delete selected item
⌘cmd + F| Search
